## ğŸŒŸ What are Primitive Streams?

> **Primitive Streams** are specialized streams designed to work with **primitive data types** directly.

Java provides three primitive stream types:

| Stream Type    | Works With |
| -------------- | ---------- |
| `IntStream`    | `int`      |
| `LongStream`   | `long`     |
| `DoubleStream` | `double`   |

### Why Primitive Streams?

* Avoid **boxing & unboxing** overhead
* Better **performance**
* Built-in numeric operations (`sum`, `average`, `min`, `max`)
